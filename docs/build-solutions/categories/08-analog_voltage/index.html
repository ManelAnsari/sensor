#Simple API

  • C/CPP
/* Add this line to your main function 
 * to read data from the sensor */
int voltage = getAdc(channel);
Note

channel - Denotes the ADC Input channel number. Typically ADC's have the ability to sense voltages from multiple sources via multiple channels. User needs to enter which channel of the ADC they have connected the voltage source.
